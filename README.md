# COBIT Quest

Um jogo de perguntas e respostas sobre Governança de TI e COBIT.

## Como Jogar

1. Abra o arquivo `index.html` em seu navegador.
2. Clique em **Novo Game**.
3. Insira os nomes das duas equipes.
4. Escolha um avatar (papel de parede) para cada equipe.
5. Responda às perguntas alternadamente.
6. A equipe com maior pontuação ao final de 40 perguntas vence!

## Estrutura do Projeto

* `index.html`: Arquivo principal.
* `css/style.css`: Estilos visuais.
* `js/script.js`: Lógica do jogo e perguntas.
* `img/`: Imagens e avatares.
